# APP FINANCEIRO (modo apresentação)

## Passo a passo

1. Abra o terminal dentro da pasta do projeto (onde está o arquivo server.js)
2. Execute o comando:
   npm install

3. Depois, inicie o servidor:
   node server.js

4. Abra o navegador e acesse:
   http://localhost:3000

---

## Login de acesso
Usuário: admin
Senha: admin

---

## O que está funcionando
- Login simulado com usuário e senha.
- Salvamento de transações em arquivo local (database.json).
- Listagem e exclusão de transações funcionando.
- Tudo é salvo automaticamente (mesmo se fechar o navegador).
